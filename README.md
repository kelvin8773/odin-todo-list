## Getting Started

To get a local copy up and running follow these simple example steps.

## Install 
```bash
git clone git@github.com:kelvin8773/odin-todo-list.git
cd odin-to-do-list
```
To install dependencies
```bash
npm install
```
To build the project
```bash
npm run build
```
To watch the changes

```bash
npm run watch
```
